# 🗣️ Perfect Pronunciation

An interactive web app that helps students practice English pronunciation with AI-powered speech recognition.

## ✨ Features
- 🎯 **Level-based learning** (A1 to C2)
- 🎤 **Real-time speech recognition**
- 🔊 **Calm voice guidance**
- 📝 **Custom word practice**
- 🎲 **Random practice mode**
